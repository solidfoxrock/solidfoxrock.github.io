Converting Character Set / Collation in MySQL Databases

Mar 25th, 2014 | Comments

Recently I got a task to convert character sets / collations of all our database tables to utf8 / utf8_general_ci. If you try googling for this topic you will see quite a lot of answers but most of them are not very accurate or completed. In this post I would like to summarize everything I gathered on the last few days about this topic. The goal is to convert all your data to use character set utf8 and collation utf8_general_ci, plus fixing all default configurations to prevent wrong character set / collation in future when creating new tables / columns.
